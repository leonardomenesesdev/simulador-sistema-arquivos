# M08 — Entrega (GitHub + PDF + AVA)

**Status:** ⏳ Pendente
**User story:** Como dupla responsável pela entrega, quero publicar o projeto no GitHub, gerar
o PDF do README e enviá-lo no AVA, para concluir o trabalho dentro do prazo (05/06).

## Descrição detalhada da feature

É a milestone de fechamento: não muda código nem documentação de conteúdo, apenas consolida e
distribui o que foi produzido. Reúne as ações operacionais finais (baseadas em
`docs/08-entrega.md`).

Passos:

- **Publicar no GitHub** — garantir que o repositório está público e atualizado (código +
  `docs/` + `milestones/` + `README.md`), com o link correto referenciado no README.
- **Gerar o PDF** — exportar o `README.md` da raiz para PDF (preview do Markdown → imprimir/
  exportar como PDF), mantendo o link do GitHub clicável/visível.
- **Revisar o PDF** — conferir seções, tabela de testes e o link.
- **Enviar no AVA** — submeter o PDF no prazo, como exige o enunciado.

Por que existe: é o ato que efetiva a nota. Tudo o que foi feito nas milestones anteriores só
conta se chegar ao AVA no formato correto.
