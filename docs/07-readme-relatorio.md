## Mini Atividades

- [ ] Criar `README.md` na raiz do projeto.
- [ ] Explicar o que e um sistema de arquivos.
- [ ] Explicar a importancia do gerenciamento de arquivos em sistemas operacionais.
- [ ] Explicar o conceito de journaling.
- [ ] Descrever tipos de journaling, como write-ahead logging e log-structured.
- [ ] Descrever as classes usadas no simulador.
- [ ] Descrever como o `.dat` armazena os dados.
- [ ] Descrever como executar o projeto em Java.

## Criterios de Aceite

- O README pode ser lido como relatorio completo.
- O link do GitHub esta visivel.
- O README explica que os arquivos simulados ficam dentro do `.dat`.
- O README pode ser impresso ou exportado em PDF.
